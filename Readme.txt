Shell Script Database Management System (DBMS):
The Project aim to develop DBMS, that enables users to store and retrieve the data from Hard-disk. 

 

The Project Features: 

The Application is CLI Menu based app, that provides to user this Menu items: 

Main Menu: 

- Create Database 

- List Databases 

- Connect To Databases 

- Drop Database 

 

Upon user Connect to Specific Database, there a new screen appears with this Menu: 

- Create Table  

- List Tables 

- Drop Table 

- Insert into Table 

- Select From Table 

- Delete From Table 

- Update Table
